Store

Store web app created on the trainee program Ruby on Rails from Applaudo Studios * ruby: 3.1.2 * rails: 6.1.7 * style: Bootstrap 4.4 * database: PostgreSQL

To run this locally, create a .env file with credentials for the postgres database

DB_USERNAME = postgres
DB_PASSWORD = admin

Run db:seed to create the items and users: | user | email | password | |———–|—————————|———-| | admin | admin@snackstore.com | 123456 | | support | support@snackstore.com | 123456 | | customer | michi@snackstore.com | 123456 |
